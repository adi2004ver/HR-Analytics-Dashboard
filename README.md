HR Analytics Dashboard (Power BI)

Project Overview

This project presents an **HR Analytics Dashboard** built using **Power BI** to analyze employee attrition and workforce patterns.
The dashboard helps HR departments understand the key factors influencing employee turnover and supports data-driven decision making.

The project visualizes employee data such as age, salary, education, job role, and years at the company to identify attrition trends.

---

Objectives

* Analyze employee attrition patterns
* Identify factors influencing employee turnover
* Provide insights for HR decision making
* Create an interactive data visualization dashboard

---

Key Dashboard Metrics

* **Total Employees:** 1302
* **Total Attrition:** 187
* **Attrition Rate:** 14.4%
* **Average Age:** 38
* **Average Salary:** 6.8K
* **Average Years at Company:** 7.4

---

Dashboard Insights

* Highest attrition occurs in employees aged **26–35**
* Employees with **salary up to 5K** show the highest attrition
* **Sales Executives** and **Laboratory Technicians** have the highest turnover
* Many employees leave within the **first few years at the company**
